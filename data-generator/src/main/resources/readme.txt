-----------
Tool usage
-----------
usage example: $ java -jar data-generator.jar -t target -r 10
Generates xls data file for Target upload with 10 random target

optional arguments:
Command     Description
-h, --help  Show this help message and exit.
-t          Type of record to create. Available values: target, s_sms, du, etisalat, phonebook, x_sms, x_voice.
-r          Total number of records to create.
