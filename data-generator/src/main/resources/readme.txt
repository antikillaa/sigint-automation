-----------
Tool usage
-----------
usage example: $ java -jar data-generator.jar -t target -r 10
Generates xls data file for Target upload with 10 random target

optional arguments:
Command	Description
-h, --help      show this help message and exit.
-t, --type      Type of record to create. Available values: target, s_sms, du, etisalat, phonebook, x_sms, x_voice.
-r, --record    Total number of records to create.
